Educational Authentication System
A Full-Stack Role-Based Authentication System built using Node.js, Express, MongoDB Atlas, JWT, and bcrypt, deployed with Render (Backend) and Vercel (Frontend).
Features:
  1.Secure User Registration & Login
  2.Password Hashing using bcrypt
  3.JWT-based Authentication
  4.Role-Based Access (Student, Teacher, Parent, Admin)
  5.Cloud Database (MongoDB Atlas)
  6.Production Deployment (Vercel + Render)
